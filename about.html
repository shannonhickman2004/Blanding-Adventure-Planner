About the San Juan County Adventure Planner

This web application helps outdoor enthusiasts find hiking trails, kayaking routes, and sightseeing locations throughout San Juan County, Utah. It combines maps, weather data, and local information in one place to make planning your next adventure easier and more fun.

Features

  • Search by location or activity type
  • Filter and sort trails based on distance, difficulty, and more
  • Get current weather and 3-day forecasts
  • View trailheads and accommodations on an interactive map
  • Save favorite adventures using localStorage
